# Project template
author: Olga Tabisheva
Points: 96
Suitcase E-Shop
Multi-page e-commerce store with HTML, SASS, and TypeScript.

Prerequisites
Node.js

Install dependencies - npm install

To run the project - npm run dev


Features
- fully Responsive - for mobile, tablet, and desktop (breakpoints: 768px, 1024px, 1440px).
- Vanilla Tech Stack
- functional E-commerce
- validation

Project Structure 
src/assets/data.json — Product data source.
src/scss/ — SASS source files.
dist/ — Compiled CSS and production assets.
